## How to install ?
 
* **Clone the repo via git**
    ```bash 
    git clone https://github.com/VladCherniavsky/InBrief-React.git
    ```
* **Enter the folder**
     ```bash
     cd Inbrief-react
     ```
* **Install dependencies**
     ```bash
     yarn install
     ```
* **Run project**
    ```bash
    yarn start
    ```
* **Visit [localhost:3000](localhost:3000)**


## To build: 

```bash
yarn  run build
```
